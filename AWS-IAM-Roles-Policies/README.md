# AWS IAM Roles & Policies Lab

## Overview
This project focuses on creating IAM users, roles, and policies using least‑privilege principles.

## Objectives
- Create IAM users and roles
- Write custom JSON policies
- Test permissions against S3/EC2

## AWS Services Used
- IAM
- S3
- EC2

## Steps Performed
- Created IAM user and role
- Attached custom policies
- Tested access restrictions
- Validated least‑privilege behavior

## What I Learned
- How IAM roles differ from users
- How JSON policies control access
- Why least‑privilege prevents breaches
