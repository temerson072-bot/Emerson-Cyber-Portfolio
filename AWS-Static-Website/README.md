# AWS Static Website Hosting (S3)

## Overview
This project hosts a static website using Amazon S3 and configures public access or CloudFront distribution.

## Objectives
- Create an S3 bucket for static hosting
- Upload HTML/CSS/JS files
- Configure bucket policy for public access
- Test website availability

## AWS Services Used
- S3
- IAM
- (Optional) CloudFront

## Steps Performed
- Created S3 bucket
- Enabled static website hosting
- Uploaded site files
- Applied correct bucket policy
- Verified site loads publicly

## What I Learned
- How S3 hosts static websites
- How IAM permissions affect deployment
- Why CloudFront improves security and performance
