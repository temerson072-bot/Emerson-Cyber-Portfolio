# AWS EC2 Security Groups Lab

## Overview
This project launches an EC2 instance and secures it using properly configured security groups.

## Objectives
- Launch EC2 instance
- Configure inbound/outbound rules
- Restrict SSH to a single IP
- Test connectivity

## AWS Services Used
- EC2
- Security Groups
- VPC

## Steps Performed
- Launched EC2 Linux instance
- Configured SSH access from my IP only
- Blocked unnecessary ports
- Verified secure access

## What I Learned
- How security groups act as firewalls
- Why open ports are dangerous
- Basics of cloud network hardening
