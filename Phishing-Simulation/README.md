# Phishing Simulation Lab

## Overview
This lab simulates a phishing attack using Phishery to inject a credential-harvesting form into a legitimate HTML page.

## Objectives
- Use Phishery to embed a credential prompt
- Host the phishing page using Apache2
- Capture credentials submitted by victims
- Analyze phishing email structure

## Tools Used
- Phishery
- Apache2 Web Server
- HTML/CSS
- Browser Developer Tools

## Steps Performed
- Injected a credential form into a cloned HTML page
- Hosted the phishing page locally
- Sent a simulated phishing email
- Captured and analyzed harvested credentials

## What I Learned
- How phishing payloads are embedded into HTML
- How attackers trick users into entering credentials
- Why email filtering, MFA, and user training matter
